# 外观定制

vnweb 提供多种外观定制选项，打造个性化使用体验。

## 主题切换

通过 `next-themes` 实现主题切换：

- **浅色模式**
- **深色模式**
- **跟随系统**

主题设置在页面顶部导航栏中一键切换。

## 自定义背景

### 背景图片

在「设置 → 外观」页面中：

1. 选择本地图片文件作为自定义背景
2. 图片自动保存到 `/assets/bg/custom/` 目录
3. 文件以时间戳命名，避免文件名冲突
4. 更换新背景时自动删除旧文件

### 游戏背景联动

支持将当前游戏封面或背景图设置为主界面背景，实现沉浸式体验。

## 本地字体

### 读取系统字体

`win/fontfile.ts` 读取 Windows 系统字体列表，供用户选择。

### 导入项目字体

`lib/font-utils.ts` 提供字体管理工具：

- 列出系统中可用字体
- 将选定字体应用到项目界面

项目默认使用 **LXGW WenKai（霞鹜文楷）** 字体。

## 图标

支持从游戏可执行文件中提取图标（`win/extract-icon.ts`），作为游戏的专属图标。

## 图片本地化缓存

通过 `/api/proxy-image/` 代理外部图片，实现图片本地化缓存与加速加载。
