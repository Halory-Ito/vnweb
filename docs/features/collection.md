# 收藏夹

收藏夹模块帮助用户对游戏进行分类整理。

## 功能概述

- 创建、编辑、删除收藏夹
- 将游戏加入/移出收藏夹
- 在首页按收藏夹筛选游戏
- 批量操作支持加入收藏夹

## 收藏夹管理

### 创建

在收藏夹管理页面，点击新建按钮，输入收藏夹名称即可创建。

### 编辑与删除

每个收藏夹支持：

- 重命名
- 删除（不删除关联的游戏）

### 游戏关联

- 在游戏详情页可以将游戏添加到某个收藏夹
- 在游戏库首页支持批量选中游戏后加入收藏夹
- 游戏可以属于多个收藏夹

## 首页集成

首页的「收藏夹」区域显示所有收藏夹，点击收藏夹可查看其中的游戏列表。

## 数据模型

收藏夹使用两张表：

**collection 表**：

| 字段        | 说明       |
| ----------- | ---------- |
| `name`      | 收藏夹名称 |
| `createdAt` | 创建时间   |
| `updatedAt` | 更新时间   |

**collection_game 表**：

| 字段           | 说明      |
| -------------- | --------- |
| `collectionId` | 收藏夹 ID |
| `gameId`       | 游戏 ID   |

两者形成多对多关系。
