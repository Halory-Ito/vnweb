# 游戏详情

游戏详情页是 vnweb 的信息中枢，集中展示单个游戏的所有相关数据。

## 页面布局

游戏详情页包含以下标签页：

### 基础信息

展示游戏的基本资料：

- 封面、Logo、背景图
- 游戏名称（原文 / 中文）
- 游戏简介
- 开发商、发行商
- 剧本、原画、音乐制作
- 游戏引擎、平台、类型
- 标签与别名

### 可编辑内容

支持直接修改：

- 基础信息（名称、简介等）
- 专属封面、背景、图标、Logo
- 可执行文件路径
- 存档目录路径

### 统计信息

- 总游玩时长
- 游玩次数
- 评分
- 最后启动时间
- 游戏状态

## 关联内容

### 角色

展示游戏关联的角色列表，点击可进入角色详情页查看角色资料（立绘、身高体重、血型、生日等）。

### PV 管理

展示游戏关联的宣传视频列表，支持在线播放。

### OST 管理

展示游戏原声带列表，支持曲目浏览与播放。

### 回忆

展示游戏的回忆内容，包含截图和 Markdown 文本记录。

### 台词摘录

展示从该游戏中摘录的经典台词，以卡片形式展示，包含台词内容、出自角色和台词背景。

## 快速操作

在详情页顶部可以：

- **启动游戏**：点击按钮直接启动游戏可执行文件
- **结束游戏**：手动标记游戏结束，记录本次游玩时长

## 技术实现

- 页面路由：`app/game/info/`
- 组件位于 `components/game/`
- 使用 React Hook Form 处理表单编辑
- 使用 TanStack Query 管理数据获取与缓存
