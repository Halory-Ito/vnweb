# 游戏库管理

游戏库管理是 vnweb 的核心模块，提供游戏的增删改查、批量操作与资料维护功能。

## 功能概述

### 首页概览

首页展示三个区域：

- **最近游玩**：显示最近启动的游戏列表
- **收藏夹**：显示用户创建的收藏夹
- **全部游戏**：以卡片网格展示所有游戏，支持分页

### 游戏列表操作

#### 排序

支持按以下维度排序：

- 游戏名称
- 添加日期
- 评分（0-10 分）
- 游玩时长
- 最后启动时间

#### 批量操作

通过多选功能，可以：

- 批量删除游戏
- 批量加入收藏夹
- 批量更新元数据

#### 筛选

- 按收藏夹筛选
- 按游戏状态筛选（未开始、游玩中、部分完成、已完成、多周目、搁置中）
- 关键字搜索（游戏名称、中文名称、别名）

## 游戏状态

每个游戏支持以下状态标记：

| 状态     | 说明           |
| -------- | -------------- |
| 未开始   | 尚未游玩       |
| 游玩中   | 正在进行       |
| 部分完成 | 已完成部分路线 |
| 已完成   | 全部通关       |
| 多周目   | 多次通关       |
| 搁置中   | 暂时搁置       |

## 数据模型

游戏的基础信息存储在 `game_info` 表中，包含以下字段：

- **基本信息**：名称、中文名、别名、简介
- **分类信息**：标签、类型、引擎、平台
- **制作信息**：开发商、发行商、剧本、原画、音乐、程序
- **媒体信息**：封面、图标、Logo、背景图
- **其他**：NSFW 标记、存档目录

## 相关页面

- 游戏详情页位于 `app/game/info/`
- 角色管理位于 `app/game/character/`
- 收藏夹管理位于 `app/game/collection/`
