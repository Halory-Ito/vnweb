# PV 与 OST 管理

vnweb 支持为游戏维护 PV（宣传视频）和 OST（原声带）资料，并提供内置播放器。

## PV 管理

### 功能

- 为每个游戏添加多个 PV 链接
- 支持在线视频播放（基于 hls.js）
- 支持本地视频文件导入

### 播放器

PV 播放使用 hls.js 实现，支持 HLS 流媒体格式。播放对话框关闭后自动停止播放并释放资源。

### 数据模型

PV 信息存储在 `game_pv` 表中，包含名称和链接地址。

## OST 管理

### 功能概览

- 为每个游戏添加多张 OST 专辑
- 每张 OST 包含多条曲目
- 支持曲目歌词管理（文本或 .lrc 文件）
- 支持多种音频格式（mp3、flac 等）
- 内置 OST 播放器

### OST 播放器

OST 播放器支持多种播放模式：

| 模式     | 说明                 |
| -------- | -------------------- |
| 顺序播放 | 按列表顺序依次播放   |
| 随机播放 | 随机选择曲目播放     |
| 单曲循环 | 重复播放当前曲目     |
| 列表循环 | 播放完列表后从头开始 |

### 数据来源

OST 资源可以来自：

- **khinsider**：从 khinsider 导入 OST 信息
- **netease**：从网易云音乐导入 OST 信息
- **手动添加**：手动输入曲目信息

### 歌词管理

- 支持纯文本歌词（`lyricsText` 字段）
- 支持 .lrc 歌词文件（`lyricsPath` 字段）
- 歌词可在播放器中同步显示

### 数据模型

OST 使用两张表存储：

**game_ost 表**：

| 字段       | 说明         |
| ---------- | ------------ |
| `name`     | OST 名称     |
| `cover`    | OST 封面图片 |
| `resource` | 资源来源     |

**game_ost_songs 表**：

| 字段         | 说明         |
| ------------ | ------------ |
| `name`       | 曲目名称     |
| `url`        | 音频链接     |
| `mediaType`  | 媒体类型     |
| `lyricsText` | 歌词文本     |
| `lyricsPath` | 歌词文件路径 |

## 相关页面

- OST 管理页：`app/ost/`
- 播放器组件位于 `app/ost/_ui/`
- PV 管理页：`app/pv/`
