# 台词摘录

台词摘录模块允许用户从游戏中摘录经典台词，并关联角色与背景信息。

## 功能概述

- 在管理页以表格形式浏览所有台词摘录
- 支持筛选、搜索和分页
- 在游戏详情页以卡片形式展示该游戏的台词
- 在角色详情页以列表形式展示该角色的台词

## 管理页面

访问路径：`/quote`

### 表格展示

台词管理表格包含以下字段：

| 字段     | 说明                   |
| -------- | ---------------------- |
| 序号     | 自动编号               |
| 游戏名称 | 台词所属游戏           |
| 台词内容 | 摘录的台词文本         |
| 出自角色 | 说这句台词的角色的名称 |
| 台词背景 | 台词出现的场景背景     |
| 创建时间 | 摘录时间               |
| 操作     | 编辑、删除按钮         |

### 表格特性

- 每个字段有**固定宽度**，超出部分显示省略号
- 鼠标悬停在省略的内容上时，通过 **Tooltip** 展示完整内容
- 底部分页组件居中显示，默认每页 10 条，可调整每页数量

### 筛选与搜索

表格顶部左侧为筛选区域：

- **按日期筛选**：选择日期范围
- **按关键字搜索**：在游戏名称、台词内容、出自角色三个字段中进行模糊匹配

### 添加台词

点击表格顶部右侧的添加按钮，弹出对话框：

1. **选择游戏**：通过可搜索的下拉选项选择游戏
2. **选择角色**：根据选定游戏自动加载角色列表（可搜索下拉）
3. **台词内容**：文本域输入
4. **台词背景**：文本域输入

> 如果选择的游戏没有关联角色，会提示"当前游戏没有关联角色，无法保存台词信息"。

### 编辑与删除

操作列提供编辑和删除两种功能，以图标按钮形式展示。

## 游戏详情集成

在游戏详情页底部的「摘录」标签页，以**卡片**形式展示该游戏的所有台词。

卡片设计：

- 左上角显示角色头像图片
- 内容以纯文本形式展示（非斜体 / 非 markdown quote）

## 角色详情集成

在角色详情页底部展示该角色的台词列表。

## 数据模型

台词摘录数据存储在 `game_quote` 表中，与 `game_info` 和 `character` 表关联。
