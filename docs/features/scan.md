# 扫描与导入

扫描模块帮助用户从本地目录批量导入游戏，支持多种扫描模式。

## 功能概述

扫描管理页面（`app/scan/`）提供以下能力：

- 维护多个扫描目录
- 选择扫描模式
- 配置扫描参数
- 查看扫描进度与结果
- 处理扫描失败项

## 扫描模式

### 层级扫描 (Mode 0)

按目录层级扫描，适合目录结构规范的游戏库：

- 可配置扫描深度（`scanLevel`）
- 扫描到指定层级后识别子目录为独立游戏
- 支持排除特定目录

### 可执行文件扫描 (Mode 1)

按可执行文件扫描，适合游戏散落在不同目录的场景：

- 递归搜索指定目录下的可执行文件
- 每个可执行文件所在的目录识别为一个游戏

## 扫描配置

### 添加扫描目录

通过系统文件选择器选择目录（非手动输入路径），添加后自动开始扫描。

### 排除目录

支持配置逗号分隔的排除目录列表，扫描时跳过这些目录。

### 扫描深度

仅在层级扫描模式下生效，控制扫描的目录层级深度。

## 扫描结果

### 扫描进度

每个扫描目录显示：

- 当前进度（0-100%）
- 已扫描到的游戏数量
- 扫描状态图标

### 失败记录

扫描失败的项目会被记录，便于后续排查与修正。

## 数据模型

扫描配置存储在 `scanner` 表中：

| 字段          | 说明                               |
| ------------- | ---------------------------------- |
| `directory`   | 扫描目录路径                       |
| `provider`    | 数据源名称                         |
| `scanMode`    | 扫描模式（0: 层级, 1: 可执行文件） |
| `scanLevel`   | 层级扫描深度                       |
| `excludeDirs` | 排除目录列表                       |
| `progress`    | 扫描进度 0-100                     |
| `gameCount`   | 已扫描到的游戏数量                 |
