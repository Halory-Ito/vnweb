# 统计与记录

vnweb 自动追踪游戏游玩数据，并提供多维度统计分析。

## 游玩记录

### 自动记录

当用户通过 vnweb 启动游戏时，系统自动开始计时：

- **开始计时**：启动游戏时记录开始时间
- **结束计时**：关闭游戏或手动结束，记录本次游玩时长
- **累计统计**：自动累加到游戏的总游玩时长

### 记录数据

每次游玩记录包含：

- 游戏 ID
- 游玩时长（秒）
- 游玩日期

## 统计视图

统计页面（`/record`）提供三种时间维度的视图：

### 周视图

- 展示本周每天的游玩时长
- 以折线图展示趋势

### 月视图

- 展示本月每天的游玩时长
- 以柱状图展示分布

### 年视图

- 展示全年各月的游玩时长
- 适合长期跟踪

## 图表类型

使用 [Recharts](https://recharts.org/) 实现两种图表：

| 图表类型 | 适用场景     |
| -------- | ------------ |
| 折线图   | 展示趋势变化 |
| 柱状图   | 展示数量对比 |

## 数据导出

支持将游玩记录导出为 Excel 格式（使用 exceljs 库）。

## 数据模型

### game_play 表

记录游戏的整体游玩状态：

| 字段             | 说明             |
| ---------------- | ---------------- |
| `gameId`         | 游戏 ID          |
| `exePath`        | 可执行文件路径   |
| `isRunning`      | 是否正在计时     |
| `totalPlayTime`  | 总游玩时长（秒） |
| `playCount`      | 游玩次数         |
| `rating`         | 评分（0-10）     |
| `lastLaunchedAt` | 上次启动时间     |
| `status`         | 游戏状态         |

### game_record 表

记录每次游玩详情：

| 字段       | 说明           |
| ---------- | -------------- |
| `gameId`   | 游戏 ID        |
| `playTime` | 游玩时长（秒） |
| `playDate` | 游玩日期       |

## 相关页面

- 统计主页：`/record`
- 周视图：`/record/week`
- 月视图：`/record/month`
- 年视图：`/record/year`
- 概览：`/record/overview`
- 导出：`/record/export`
