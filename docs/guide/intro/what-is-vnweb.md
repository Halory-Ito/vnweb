# 概述

vnweb 是一个本地游戏管理软件，具备 Web UI 和 CMD ，支持多刮削器、账号绑定、游玩统计、数据可视化、各种游戏垂类信息、主题设置、备份恢复等功能

> 日常管理用 Web UI，快速启动游戏用 CMD

多的这里暂且不说，先看看整体效果吧

# 缘起

主播是在2024年入坑的视觉小说，第一部是《饿殍》，推了70%吧，但是由于一些原因，耽搁了好久，后来就没有玩了

接着是《fsn》，情况类似，因为没有看攻略，也没有频繁save的习惯，推到后面好像只剩下be了，也没有玩下去了

然后是《素晴日》，久仰大名哇，但是**当时**序章还没推完呢，感觉一般吧，也没有继续推下去了

第一部推完的应该是《弹丸论破1》，感觉很nice，画风也很有特色，这时才感受到视觉小说的魅力

于是陆陆续续推完了《素晴日》、《弹丸论破2》、《魔法使之夜》等有口碑的视觉小说，一发不可收拾

# 为什么要做这个

随着本地视觉小说的资源越来越多，管理起来有点不够优雅，因为面对的全是文件夹，和视觉小说画风不符

一开始我是使用的其他视觉小说管理软件，先是PotatoVN，起初我是震惊的，原来还有这种软件，而且用起来还方便，但缺点就是可定制化太低了，我对样式有点挑剔

后来接触了Vnite，这个就好看很多了，并且支持的功能也非常丰富，但是还是不够，如果要对视觉小说进行管理的话，那就应该尽量支持其各种垂类信息，恨不得做成wiki的程度；此外，也存在几个小bug，让我有点接受不了

因此，一方面对开发这类项目有兴趣，另一方面要搞毕设了，而毕设选题我可以自己决定，于是就开整了
